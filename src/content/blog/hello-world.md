So I made a website. Not sure why it took me this long.

I left Trata at the start of this year. We co-founded it in late 2024, and I was CTO. It was a wild ride, but it was time to move on. No hard feelings — just needed to do my own thing.

Right now I'm building [orb-ui](https://github.com/alexanderqchen/orb-ui), an open-source React component library for voice AI agents. The idea is simple: voice AI is going to be everywhere, and the UI layer for it barely exists. I want to build the standard.

This site is my home on the internet. I'll write about what I'm building, what I'm learning, and whatever else feels worth sharing. No content strategy. No SEO optimization. Just honest writing.

If you want to follow along, I'm [@alexanderqchen](https://twitter.com/alexanderqchen) on most platforms.

Let's see where this goes.
